# Hotel Reservation System

## Overview

The Hotel Reservation System is a Java console-based application that allows users to search, book, and manage hotel room reservations. It is developed using Object-Oriented Programming (OOP) concepts and File Handling for storing booking data.

## Features

* Search available hotel rooms
* Room categories (Standard, Deluxe, Suite)
* Book hotel rooms
* Cancel reservations
* Payment simulation
* View booking details
* Store bookings using File I/O (`bookings.txt`)

## Technologies Used

* Java
* Object-Oriented Programming (OOP)
* File Handling (File I/O)
* VS Code

## How to Run

1. Compile the project:

```
javac *.java
```

2. Run the program:

```
java Main
```
